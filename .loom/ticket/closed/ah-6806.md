# Collapsible entire Layers pane

The Layers pane was incorrectly made with collapsible legend sections (Road Access, Land Ownership). The entire pane should collapse instead, like the WaypointPanel pattern - showing only header when collapsed.

## Design

1. Remove CollapsibleLegend component and per-section collapse state
2. Add collapsed prop/state to LayerPanel (like WaypointPanel)
3. Add collapse button in header
4. Hide all layer toggles and legends when collapsed, show only header
5. Update App.css for collapsed/expanded states

## Acceptance Criteria

Layers pane has collapse button in header. When collapsed, only 'Layers' header is visible. When expanded, all layer toggles and legends are visible. Individual legend sections are NOT collapsible.
