# Stack collapsed waypoint/path panels vertically

The waypoint panel and path panel overlap when both are collapsed at the bottom-left. They should be stacked vertically with clear spacing.

## Design

In App.css, set different bottom values for collapsed panels:
- .waypoint-panel.collapsed: bottom: 50px (current)
- .path-panel.collapsed: bottom: 100px (stack above waypoint)

This ensures both collapsed panels are visible and clickable without overlap.

## Acceptance Criteria

Waypoint panel at 50px from bottom. Path panel at 100px from bottom. No overlap when both collapsed. Both panels clickable.
