# Scale bar: yards/miles only

The scale bar currently shows feet/miles (imperial). It should only show yards or miles. For distances under 1760 yards, show yards. For 1760+ yards, show miles with 1 decimal.

## Design

MapLibre ScaleControl only supports imperial (feet/miles) or metric. Need custom control:
1. Create custom MapLibre IControl implementation
2. Listen to map zoom/move events
3. Calculate scale from map.getBounds() and container width
4. Format: yards if < 1760, otherwise miles (1 decimal)

## Acceptance Criteria

Scale bar shows only yards (e.g., '500 yd') or miles (e.g., '1.5 mi'). Never shows feet.
