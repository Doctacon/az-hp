# Measurements in yards/miles

Distance measurements (measure tool and path drawing) currently show feet for small distances and miles for larger. All measurements should be in yards or miles only: yards for < 1760, miles (1 decimal) for 1760+.

## Design

Update formatting in useMeasurement.js, MeasurePanel.jsx, usePathDrawing.js, PathPanel.jsx:
const METERS_TO_YARDS = 1.09361
const YARDS_PER_MILE = 1760

function formatDistance(meters) {
  const yards = meters * METERS_TO_YARDS
  if (yards < YARDS_PER_MILE) {
    return `${Math.round(yards)} yd`
  }
  return `${(yards / YARDS_PER_MILE).toFixed(1)} mi`
}

## Acceptance Criteria

Measure tool shows yards (e.g., '750 yd') or miles (e.g., '2.3 mi'). Path panel shows distance in yards or miles. Never shows feet.
