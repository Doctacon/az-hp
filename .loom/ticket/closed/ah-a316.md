# Freehand path drawing with GeoJSON export

Draw mode to create paths by clicking vertices. Users can name paths, see total distance, and export as GeoJSON LineString for use in other apps (GPS devices, other mapping tools).

## Design

1. Add Draw Path button to toolbar
2. Create usePathDrawing hook with:
   - paths array (each path: coordinates + name + metadata)
   - currentPath (in-progress path)
   - addVertex(), finishPath(), deletePath(id)
3. Add map click handler when draw mode active
4. Render path line + vertex markers on map
5. Create PathPanel component to list saved paths:
   - Path name (editable)
   - Total distance
   - Export button (GeoJSON download)
   - Delete button
6. Persist paths to localStorage (survive page refresh)
7. Export generates valid GeoJSON FeatureCollection with LineString features

## Acceptance Criteria

Draw Path button toggles draw mode. Clicking adds vertices, shows path line with distance. Double-click or Finish button ends path. Paths listed in PathPanel with editable name and distance. Export button downloads valid GeoJSON LineString file. Paths persist across page refresh via localStorage.
