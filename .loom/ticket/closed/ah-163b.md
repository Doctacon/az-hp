# Labeled land features from Overture

Add comprehensive labels for topographic features (peaks, valleys, ridges, canyons), water features (lakes, rivers, springs, tanks), and road/trail names from Overture Maps. Currently the map has very few labeled features compared to gohunt.

## Design

Pipeline changes (pipeline/02_clip_arizona.py):
- Extract names from Overture base layer for topo features
- Include water feature names from water layer
- Ensure road/trail names are preserved in roads layer

Tile generation (pipeline/04_generate_tiles.sh):
- Include 'name' attribute in tile generation for all relevant layers

Frontend (frontend/src/config/layers.js):
- Add symbol layers for each label type with appropriate styling
- Set minzoom thresholds (labels appear at zoom 10+)
- Use text-halo for readability

## Acceptance Criteria

Peak/summit names visible at appropriate zoom. Lake and river names appear. Forest road numbers (FR 123) visible. Labels don't overlap excessively. Styling matches existing map aesthetic.
